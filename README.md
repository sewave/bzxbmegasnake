# Mega Serpiente

For the **ZX Spectrum** using **Boriel ZX Basic**.

Special thanks to DamienG for the [ZX Origins](https://damieng.com/typography/zx-origins/) awesome fonts and Andrew C. E. Dent for the [8x8.me](https://github.com/ace-dent/8x8.me) characters.

Using Koncrete font (https://damieng.com/typography/zx-origins/koncrete/).

Built using ZXBasic Studio.

Build with -D DEBUG to enable level skip, -D DEBUG_BORDERS to enable time consumption of different functions and -D LANG=en to enable english.

Wave 2025
